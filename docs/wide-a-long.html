6 Wide a long

pivot_longer

“Wide” -> “long”

soja_long <- soja %>% 
  pivot_longer(
    cols = starts_with("bk"),
    names_to = "bk",
    names_prefix = "bk_",
    values_to = "rinde"
  )

soja_long

Hagamos un último control de calidad…

str(soja_long)

soja_long <- soja_long %>% 
  mutate_at(vars(trt, bk), as.factor)

soja_long

Ahora si, listo para un primer gráfico (por el momento no importa la apariencia)…

soja_long %>% 
  ggplot() + 
  aes(x=trt, y =rinde) + 
  geom_point(alpha=0.2)
